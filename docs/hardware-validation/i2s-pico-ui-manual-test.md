状态：待执行

证据等级：未填真实硬件记录前只能作为 fixture / 软件手工测试候选

## 1. 测试目标

验证 VSCode 扩展打开真实 `.lac` 后，右侧协议解码面板可以选择 I2S，配置 SCK/WS/SD 映射、字长和对齐方式，并通过 `runDecoder('i2s')` 输出左右声道 annotation 结果。

本模板不覆盖 binary 输出或 WAV 导出。当前 I2S 只声明 annotation 样本输出，binary/WAV 能力标记为不支持。
