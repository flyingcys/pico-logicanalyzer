# 功能状态矩阵

更新时间：2026-04-28

本矩阵用于约束文档声明，避免把占位、模拟或实验性能力描述为生产可用能力。状态定义如下：

| 状态 | 含义 | 发布说明要求 |
| --- | --- | --- |
| 已实现 | 有源码主链路、自动化测试或可复现手动验收记录支撑 | 可以写入主要功能说明 |
| 实验性 | 有主链路或框架，但缺少真实硬件、真实样本或跨平台验证 | 必须标注实验性和限制 |
| 模拟 | 依赖 mock、示例数据、固定返回值或占位 UI | 不得写成真实硬件能力 |
| 规划中 | 尚未接入主链路 | 只能出现在路线图或待办 |
