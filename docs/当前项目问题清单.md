# 当前项目问题清单

基于代码审查（不依赖文档），以下是当前项目中存在的主要问题和待修复项。已按优先级排序。

## 1. 类型安全问题 (High Priority)
- 276 个 any 类型使用（严格模式要求 noImplicitAny）
  - 主要位置：src/extension.ts, src/utils/DecoderBenchmark.ts, src/tools/TypeScriptCodeGenerator.ts, src/tools/PythonDecoderAnalyzer.ts 等
- 违反 strict TS 配置：any 断言、参数和返回类型使用 any

## 2. TODO 和 FIXME 标记
- 14 个 TODO 标记：
  - src/tools/TypeScriptCodeGenerator.ts: 解码器生成 TODO
  - 其他代码中的临时注释
- 需要清理或实现

## 3. 资源泄漏风险
- 57 个资源泄漏检测：
  - setInterval / setTimeout 未 clear
  - 可能未释放的定时器、监听器
  - 主要在 utils/DecoderBenchmark.ts, services/ 等

## 4. 协议解码器问题
- 部分协议解码器实现不完整或有 bug（I2C, SPI, UART 等）
- StreamingDecoder 和 DecoderBase 接口使用不一致

## 5. 驱动层问题
- SaleaeLogicDriver.ts 等驱动文件存在 setTimeout 和性能问题
- 多设备支持（MultiAnalyzerDriver）未充分测试

## 6. 前端 UI 和 Webview 问题
- Vue3 组件中可能存在的类型错误、响应式问题
- 一些 HTML 残留或迁移不完全

## 7. 测试覆盖不足
- 部分单元测试未覆盖错误路径、边缘情况
- 性能和压力测试部分依赖 mock

## 8. 数据库和配置问题
- DatabaseManager 中的同步问题
- ConfigurationManager 中的缓存同步
