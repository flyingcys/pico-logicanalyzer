## 更新规则

- 新增能力必须同步更新功能状态矩阵或专题文档。
- 发布说明不得使用“正式版本”“完整支持”“100% 兼容”等措辞，除非 release gate 和认证矩阵有证据。
- 历史分析文档保留原始内容时，必须在索引中标注状态，避免与当前口径混用。
